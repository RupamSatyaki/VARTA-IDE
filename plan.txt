  VARTA вЂ” Desktop Code Editor
  Full Build Plan (Updated)

TECH STACK
----------
- Electron (main process)
- Vite + electron-vite (build tooling)
- React 18 + TypeScript
- Monaco Editor (@monaco-editor/react)
- Zustand (state management)
- Tailwind CSS (styling + CSS variables for theming)
- xterm.js (terminal renderer)
- node-pty (PTY backend)
- simple-git (git operations)
- chokidar (file watching)
- electron-builder (packaging)

CHANGES FROM ORIGINAL PLAN
---------------------------
1. Phase 16 (UI primitives) moved to before Phase 5 в†’ now Phase 5
2. src/shared/errors.ts added in Phase 2 for typed IPC error codes
3. AI Panel phase added after Phase 15 (now Phase 16):
   - aiStore, aiHandlers, AIChatPanel, AIInlineHint
   - Claude API integration via IPC
   - API key stored in settings, NEVER exposed to renderer directly
